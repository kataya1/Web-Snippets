This is a demo using while loops no real fetching

This simulate what if javascript was Synchronous aka blocking

Try clicking the random color before and after clicking run

Everything else like events is queued while the browser is fetching

Ofcourse events and others are registered but don't take effect except after the code executes

let foo, bar, baz;
fetch("fiveSecondRequest.com\hello").then( data => foo = data.response )
fetch("fiveSecondRequest.com\Greetings").then( data => bar = data.response )
fetch("fiveSecondRequest.com\Whatever").then( data => baz = data.response )

console.log(foo)
console.log(bar)
console.log(baz)
